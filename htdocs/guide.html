New Session User Guide

This page gives step-by-step instructions on how to conduct a new session

Step 1: Enter Course Information

  • Course Code Format: XXXX1234

Step 2: Enter Session Information

  • Class Length needs to be divisible by Interval Length
  • Click "Is Groupwork involved" for more options to appear

Step 3: Click on Codes that you observe happening

  • Hover over codes to see what they mean
  • Alternatively, use the "COPUS Code List" button under the timer to see all code information
  • Click on a square to record it. It will appear green after being clicked
  • Click on a green square to un-record the square

Step 4: Record Engagement by Clicking

  • Select the engagement level, and what students are engaged with

Step 5: Record any Custom Notes (Optional)

  • Optional
  • Record observations that cannot be accurately described by COPUS codes.
  • Can be helpful for later reference

Step 6: Wait for Interval to Finish

Note: If the class finishes early, click the "Finish Session Early" button.
